# Base2 文件加密器

## &#x20;项目简介

这是一个基于 **Python tkinter** 开发的 **Base2（二进制）文件加密/解密工具**，提供直观的图形界面，支持将任意文件或文本转换为二进制字符串（由 0 和 1 组成），以及将二进制数据还原为原始文件。

***

## 功能特性

### 加密功能

- **文件模式**：选择任意文件，将其内容转换为二进制字符串
- **文本模式**：直接输入文本，转换为对应的二进制编码
- **流式处理**：采用分块读取和写入，支持大文件处理（已测试 100MB+ 文件）
- **实时预览**：显示前 100 位二进制数据，方便确认转换结果
- **智能导出**：自动生成带时间戳或原文件名的输出文件

### 解密功能

- **文件输入**：选择包含二进制数据的 `.txt` 文件进行解密
- **文本输入**：直接粘贴或输入二进制字符串进行解码
- **自动清理**：自动过滤空格、换行符等无关字符
- **多编码支持**：优先使用 UTF-8 解码，失败时自动降级到 Latin-1

### 界面特性

- **标签页布局**：加密/解密功能分页展示，清晰直观
- **实时统计**：显示总位数、已处理位数、剩余位数、处理速度
- **进度条反馈**：大文件处理时显示实时进度百分比
- **状态提示**：成功/错误状态用颜色区分（绿色/红色）
- **剪贴板支持**：小文件可一键复制结果到剪贴板（≤1KB）

### 性能优化

- **多线程架构**：后台线程处理耗时操作，界面不卡顿
- **内存优化**：流式读写，避免一次性加载大文件到内存
- **智能更新**：小文件瞬间完成，大文件才启用进度条更新

## 使用说明

### 加密流程（文件 → 二进制）

1. 打开程序，默认在「加密」标签页
2. 选择输入方式：
   - **选择文件**：点击"选择文件"按钮，选取要加密的文件
   - **输入文本**：切换到"输入文本"模式，在文本框中输入内容
3. 点击 **"转换后导出为文档"** 按钮
4. 选择保存位置和文件名
5. 等待处理完成，查看预览区和统计信息

### 解密流程（二进制 → 原始数据）

1. 切换到「解密」标签页
2. 选择输入方式：
   - **选择二进制文件**：点击按钮选择之前生成的 `.txt` 文件
   - **手动输入**：在文本框中粘贴二进制字符串
3. 点击 **"导出解密为文档"** 按钮
4. 选择输出路径和文件名
5. 完成后即可获得原始文件

### 小技巧

- **快速复制**：对于小文件（≤1KB），可直接点击"复制到剪贴板"
- **大文件警告**：超过 100MB 的文件会弹出确认提示
- **文件名智能识别**：解密时自动去除 `.txt` 后缀作为默认输出名
- **实时验证**：输入内容变化时自动检测有效性

没有更多了=D

**核心类**：`Base2Encryptor` (继承自 `tkinter.Tk`)

**主要方法**：

- `file_to_binary()` - 文件转二进制（流式处理）
- `text_to_binary()` - 文本转二进制
- `binary_to_bytes()` - 二进制转字节
- `_streaming_decrypt_file()` - 流式解密文件

## &#x20;注意事项

1. **非专业加密工具**：这只是编码转换，不具备真正的加密安全性
2. **大文件体积膨胀**：二进制格式会使文件大小膨胀约 **8 倍**（1 字节 → 8 字符）
3. **内存限制**：虽然支持流式处理，但极端大的文件仍需注意系统资源
4. **编码问题**：特殊字符可能在不同系统间存在兼容性差异
5. **仅限学习娱乐**：请勿用于正式的安全场景！
